          SOBRE SUPERSPACE 3 🚀
  Se trata de um jogo de simulação espacial, tem o seguinte:
    - planetas:Com cores procedurais,aneis,tipos,colônias(não garantida que sejam humanas :) ),atmosfera, e satélites naturais.🪐
    - estações espaciais:Se localizam próximas de planetas, nela pode acoplar para comprar ou vender produtos e computadores para movimentação rá-
    pida.🤖
    - estrelas:Podem ser utilizadas para localizar um sistema solar, não recomendaria tentar pousa-los >:).☀️
    - movimentação rápida:No jogo existem 2 tipos de saltos, que são:'Salto espacial':Serve para movimentação entre planetas.;
    'Salto hypespacial':Serve para movimentação entre sistemas, requer computador especial que pode ser comprado nas estações, e não é barato.🥏
  Sobre a movimentação manual, para rodar a câmera usa-se A e D, para movimentar a nave na direção escolhida use O para ligar o motor, e P para
desligar, para mover para cima W e para baixo S, desculpa se esse W e S serem confusos e odiosos mas foi por que não consegui criar um bom sis-
tema de câmera com eixo Y no meio, para aumentar a velocidade R e para diminuir T.
  Se for testa-lo deve ter PyOpengl instalado ,o seu acelerador e o glfw, recomendaria pesquisar sobre no Youtube para sua instalação.Desejo que
goste do meu projeto claro não espere gráficos incríveis pois como já dito tenho 12 anos, obrigado por ler isso.😁
